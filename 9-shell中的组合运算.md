### &&、|| 并列写法
```bash
echo xx && { echo yy || echo zz; }  #注意分号的用法
```
- 除了{左边必须要有一个空格,其他地方都可以不用空格
- zz后边的分号不能省略
----

### []判断中的结合写法
```bash
[ 1 -lt 2 -a \( 3 -lt 0 -o 2 -lt 3 \) ] && echo ok
```
- 括号改变结合顺序
- 注意括号要转义
----

### 优先级(高-->低): -a 、 -o 、 ! 
```bash
[ ! '' -o '1' ] && echo ok || echo xx    ==> xx
```
----

#### [[]]的写法
```bash
[[ "${num}" =~ "^[0-9]*$" ]]
[[ $ak > 5 || $ak< 9 ]]
```
- [[中间不可有空格
- 未找到-o  -a的写法, 用&& || 代替
